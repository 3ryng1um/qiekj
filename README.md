### 这啥
胖乖生活（仅饮水）跳过广告的脚本？请求是抓包抓出来的，本人不会脱360加固的壳。

我把自用的版本拿ai改了一下，有需要可以自行修改。不想用脚本也可以下个http快捷方式然后按照这个流程封装成按钮，用的时候按一下解锁就行了
### 能干什么：
1. 取水不用扫码，不会有乱七八糟的广告弹窗 
2. 或许新用户可以不用绑定支付宝直接用，但订单还是要付款的 
3. 里面有个自动刷积分的功能（知乎上大佬写的，不怕封号可以试试）
4. 至少抓包出来的链接和设备skuid,imei啥的还是挺全的，可以随便用来干点啥

### 怎么用
手机上下个能运行python的玩意儿，装两个库。

填入自己需要的shopID，或者用选择最近使用过的设备

有token直接把token填里，没有就运行一下login函数然后接验证码，脚本会把token打印出来
